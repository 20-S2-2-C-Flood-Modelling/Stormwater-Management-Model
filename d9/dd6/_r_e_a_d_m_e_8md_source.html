16 # Introduction
17 This is the open source SWMM source code repository maintained by the Open
18 Water Analytics group.
19 
20 SWMM is a dynamic hydrology-hydraulic water quality simulation model. It is
21 used for single event or long-term (continuous) simulation of runoff quantity
22 and quality from primarily urban areas. SWMM source code is written in the C
23 Programming Language and released in the Public Domain.
